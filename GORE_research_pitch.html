Research Proposal

GORE
Graph Of
Recursive Execution

A minimal Turing-complete language designed as a controlled training environment for backtracking reasoning in small LLMs. Zero pretraining leakage. Infinite synthetic data. Verifiable ground truth.

Spectral-R1 ablation candidate · NeurIPS 2026
02 / 06
The Problem

We can't study reasoning in a clean room

When you train an LLM to reason on Python or natural language, you can't tell if it learned to reason or just memorized pretraining patterns.

01
Pretraining leakage
Models already know Python. Any capability could be retrieval, not reasoning.
02
No dense reward
CoT is unstructured text. No intermediate verifiable steps between prompt and answer.
03
OOD unclear
Hard to define out-of-distribution for natural language tasks.
04
Backtracking implicit
Search is hidden inside the model. Nothing to inspect or supervise directly.
03 / 06
The Language

Three primitives. Full Turing power.

GORE programs are explicit search trees. Every execution is a verifiable, inspectable graph of decisions.

# STEP — deterministic unification X = red # FORK — all branches live simultaneously ? { X = a | X = b | X = c } # CUT — prune branch on unification failure ! X = goal -> search
FORK (3 branches) branch[0] → STEP X = red ✓ branch[1] → STEP X = green ✓ branch[2] → CUT failed → backtrack

Core Hypothesis

A micro-LLM trained on GORE traces will show better OOD generalization on unseen tree depths than a larger model trained on natural language CoT — because the inductive bias of the language matches the inductive bias of the task.

05 / 06
Experiments

Four clean ablations

All experiments run at micro-LLM scale (GPT-2). Curriculum over 5 levels of tree complexity.

E1
SFT on traces
Train on (program, query) → trace → solutions. Measure OOD at depth=4, width=5.
E2
GRPO + interpreter reward
Use gorevm as verifier. Dense reward on each step. Compare to E1.
E3
Spectral analysis
Measure activation spectra during FORK vs CUT vs STEP. Clean env → clean signal.
E4
Transfer to NL
Does GORE-trained backtracking transfer to natural language multi-hop QA?
